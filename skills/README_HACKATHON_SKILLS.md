# 黑客松 Skills 说明

本产品已**完全特化为黑客松模式**，仅使用以下 Skills 文件。

## 总监督（参与所有交互）

| 文件名 | 说明 |
|--------|------|
| `hackathon_supervisor.md` | 程序员总监督：参与**所有** 6 大分区与 AI 的交互上下文，可写评审标准、表述规范、技术严谨性等。 |

## 六分区（各自主导其分支）

每个分区下的追问、Tips、回答与补全，会注入**总监督 + 该分区文件**的内容。

| 文件名（id） | 分区主题 |
|-------------|----------|
| `hackathon_problem.md`  | 提出问题与机会 |
| `hackathon_solution.md` | 解决方案 |
| `hackathon_tech.md`     | 技术实现 |
| `hackathon_business.md`| 商业模式 |
| `hackathon_team.md`    | 团队与执行 |
| `hackathon_risk.md`    | 风险与应对 |

**命名规范**：总监督固定为 `hackathon_supervisor.md`；分区为 `hackathon_<分区英文名>.md`。  
直接编辑对应 `.md` 保存即可生效。
