## 使用说明

- 新人上手：先读 **README.md**（项目根目录）与 **PRD**，再按需查阅 **ARCHITECTURE**、**DECISIONS**。
- 开发时：接口与数据模型以 **PRD** 与后端代码为准；前端模块结构见 **ARCHITECTURE**。
- 策划 / 运营：**宣传片策划** 用于沉淀分镜、文案与排期。

若新增文档，建议在本索引中补充一行，并保持 PRD 与代码行为一致。
